# 🛰️ Battlefield Awareness System

An advanced web-based Battlefield Awareness System designed to provide real-time situational awareness using geospatial visualization, intelligent risk analysis, and AI-driven decision support.

The project combines interactive 3D mapping, dynamic unit tracking, simulated real-time battlefield data, and intelligent algorithms into a unified monitoring platform for enhanced operational awareness and decision-making.

---

# 📌 Features

- 🌍 Interactive 3D Battlefield Map
- 📍 Real-Time Unit Tracking
- 🔥 Dynamic Heatmap Risk Visualization
- 🧠 AI-Based Threat Analysis
- 🚨 Priority-Based Alert System
- 🛣️ A* Pathfinding Navigation
- 📊 Treemap Risk Breakdown
- ⚡ Real-Time Simulation Updates
- 🧩 Modular Scalable Architecture
- 🎯 Responsive Modern Dashboard UI
